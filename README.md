## 关于 OPC

OPC (One Person Company) 是一种新型的商业形态——一个人也能构建有价值的商业，创造有意义的产品。

OPC Founder 是我的个人网站，记录和展示一人公司模式的探索之旅。

## 我在做什么

### AI Agent 开发

**OpenClaw** - 正在研发的 AI Agent 项目，探索大语言模型在实际业务中的应用。

### 内容创作

分享技术文章、创业心得与个人成长，帮助更多人了解一人公司的可能性。

### 产品开发

从想法到实现，独立完成产品的设计、开发与运营。
